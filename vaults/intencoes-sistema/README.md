# Vault de Intenções

Bem-vindo ao seu Obsidian Vault para gerenciar intenções do sistema de busca inteligente!

## 📁 Estrutura

- **Intenções/** - Suas definições de intenções organizadas por categoria
  - **Saudações/** - Cumprimentos e saudações
  - **Suporte/** - Respostas de suporte e ajuda
  - **Informações/** - Informações gerais
  - **Ações/** - Ações e comandos
  - **Análise/** - Análise e insights
- **Templates/** - Templates para criar novas intenções
- **Análise/** - Análises de queries não respondidas
- **Logs/** - Histórico de mudanças

## 🚀 Como Usar

1. Crie suas intenções em `Intenções/`
2. Use o template em `Templates/` para padronizar
3. Vincule intenções relacionadas com `[[Link]]`
4. Use tags `#categoria` para organizar

## 📝 Formato de Intenção

```yaml
---
id: meu_id_unico
triggers:
  - palavra-chave-1
  - palavra-chave-2
resposta: "Sua resposta aqui"
prioridade: 10
---

# Título da Intenção

Descrição detalhada...
```
